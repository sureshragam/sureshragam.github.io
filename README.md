# 🚀 Suresh Ragam — Futuristic Developer Portfolio

A modern futuristic developer portfolio built using React, TypeScript, Framer Motion, Redux Toolkit, and modular component architecture.

This project is designed as more than just a personal portfolio — it is an interactive frontend experience inspired by futuristic interfaces and the long-term vision of building a personal AI assistant ecosystem similar to Jarvis.

## ⚡ Performance Optimizations

The project has been optimized heavily for smoother rendering and better Lighthouse scores.

### Optimizations Included

- Lazy loading with `React.lazy`
- Intersection Observer based rendering
- Reduced unnecessary rerenders
- `React.memo` optimizations
- Optimized mouse tracking using `requestAnimationFrame`
- Reduced animation complexity
- Deferred heavy visual effects
- Image optimization and lazy loading
- Scroll performance improvements
- GPU-friendly animations using `transform`

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/sureshragam/sureshragam.github.io.git
```

## Navigate Into Project

```bash
cd sureshragam.github.io
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm start
```
